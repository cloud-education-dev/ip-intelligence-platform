# Notion Documentation

## Product Vision
Create an open-source, visual, educational networking lab for IP addressing, CIDR, routing, AWS networking and troubleshooting.

## Personas
- Beginner learning subnetting
- Cloud engineer designing VPCs
- Network engineer validating route plans
- Interview candidate practicing concepts
- Instructor building labs

## Definition of Done
- Pure domain logic tested
- Interactive UI with accessible labels
- Exports work offline
- Docs and deployment ready
- Examples teach AWS/Cisco/Linux equivalents
