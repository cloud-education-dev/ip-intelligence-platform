# Development Roadmap

## Phase 1 — MVP Foundation
- Calculators, validation engine, VPC graph, IPAM store, tests, Docker deployment.

## Phase 2 — Designer UX
- Drag/drop palette, dockable panels, command palette, undo/redo for canvas, keyboard shortcuts.

## Phase 3 — Simulation
- Packet timelines, ARP/NAT/firewall state machines, route tracing, troubleshooting root-cause engine.

## Phase 4 — Education Platform
- Guided labs, quizzes, interview mode, Cisco/Linux/AWS equivalence examples.

## Phase 5 — Enterprise/Open Source
- Plugin API, optional backend sync, collaboration, import from Terraform/AWS, Containerlab runner adapter.
